Redstoner - ModuleLoader

Redstoner's ModuleLoader allows Spigot servers to load, reload, and unload modules without the need to restart the server or do the troublesome /reload command. Modules are basically mini-plugins. They have some ModuleLoader specific features, and can do anything plugins can do, but as a guideline, modules tend to be small.

Redstoner - Modules

This repo contains all the modules Redstoner uses on the server. Right now there are 42 modules currently running. While some modules do talk to each other, either directly or indirectly, each module is intedpendent of each other. The module DataManager is required by a majority of modules though.
